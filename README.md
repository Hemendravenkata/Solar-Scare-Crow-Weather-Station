# Solar-Scare-Crow-Weather-Station
This project helps farmers reduce crop damage caused by birds while also providing important weather information for better farming decisions.
Solar Scarecrow and Weather Station
Project Overview

The Solar Scarecrow and Weather Station is an IoT-based agricultural project designed to protect crops from birds and monitor environmental conditions. The system uses solar energy to power a scarecrow mechanism that detects bird movement and scares them away. It also collects weather data such as temperature, humidity, and pressure to help farmers make better decisions.

Features:
Solar powered system
Bird detection using PIR motion sensor
Sound alert using speaker module
Weather monitoring system
Real-time environmental data colection

Technologies Used:
Arduino
IoT Sensors
Embedded Systems
Solar Energy 

Components:
Arduino Microcontroller
DHT22 Temperature & Humidity Sensor
UV Light Sensor
BMP180 Pressure Sensor
PIR Motion Sensor
Speaker Module
Solar Panel
DC Motor

How the System Works:
PIR sensor detects bird movement in the field.
The scarecrow activates sound or movement to scare birds away.
Solar panel powers the system.
Weather sensors collect environmental data.

Applications:
Smart agriculture
Crop protection
Weather monitoring

Future Improvements:
